Oque aprendemos essa semana?
O HTML (HyperText Markup Language) é uma linguagem de marcação utilizada para estruturar páginas web. Dentro do HTML, o BODY representa a parte visível do site, enquanto o HAEd contém informações de funcionamento e configuração da página.

O CSS (Cascading Style Sheets) é a linguagem responsável pela estilização das páginas web, definindo cores, tamanhos, fontes, posicionamento e aparência visual dos elementos.

O JavaScript (JS) é uma linguagem de programação utilizada para adicionar funcionalidades e interatividade aos sites. No JavaScript aprendi sobre variáveis, como:

const: variável de valor imutável;
let: variável de valor mutável.

Também vimos operadores matemáticos, como:
+ (adição)
- (subtração)
* (multiplicação)
/ (divisão)
% (resto da divisão)
** (exponenciação)
++ (incremento)
-- (decremento)

Além disso, vi os tipos de dados:

String: sequência de caracteres;
Int: números inteiros;
Float: números decimais ou (números quebrados);
Boolean: valores verdadeiro (true) ou falso (false).

No Linux, aprendemos que é necessário conhecer comandos básicos para realizar tarefas no sistema, sendo cada comando responsável por uma função específica. Alguns exemplos são:

cd: acessar uma pasta;
ls: listar arquivos e pastas;
mkdir: criar pastas;
touch: criar arquivos.
